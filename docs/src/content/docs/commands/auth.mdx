## login

Start an interactive login wizard:

```bash
redmine auth login
redmine auth login --name work   # explicit profile name
```

| Flag | Description |
|------|-------------|
| `--name` | Profile name (default: derived from server hostname) |

<Aside type="tip">
  Run `redmine auth login` again at any time to add another profile — the wizard never overwrites an existing one without confirmation.
</Aside>

## list

```bash
redmine auth list
```

Shows each profile's server URL and which one is currently active.

## switch

```bash
redmine auth switch
```

Opens an interactive selector. The picked profile becomes the default for subsequent commands.

## status

```bash
redmine auth status
```

Displays the active profile name, server URL, and authentication method.

## logout <Badge text="destructive" variant="caution" />

```bash
redmine auth logout              # current profile
redmine auth logout work         # named profile
```

You will be prompted to confirm before removal.

<Aside type="caution" title="API keys in backups">
  After logout, revoke the corresponding API key in Redmine if it may exist in shell history or backups.
</Aside>
