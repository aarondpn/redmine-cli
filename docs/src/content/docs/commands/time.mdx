## log

```bash
redmine time log [flags]
```

Without flags, opens an interactive form.

| Flag | Description |
|------|-------------|
| `--issue`    | Issue ID |
| `--project`  | Project (name or identifier) |
| `--hours`    | Hours spent — <Badge text="required" variant="danger" /> |
| `--activity` | Activity type (name or ID) |
| `--comment`  | Description of work |
| `--date`     | Date (YYYY-MM-DD or `today`, default: today) |

<Tabs syncKey="time-log">
  <TabItem label="Interactive">
    ```bash
    redmine time log
    ```
  </TabItem>
  <TabItem label="One-shot">
    ```bash
    redmine time log --issue 123 --hours 1.5 --activity Development --comment "Bug fix"
    ```
  </TabItem>
</Tabs>

## list

```bash
redmine time list [flags]
```

| Flag | Description |
|------|-------------|
| `--project`    | Filter by project |
| `--user`       | Filter by user: `me`, name, or ID |
| `--issue`      | Filter by issue ID |
| `--activity`   | Filter by activity (name or ID) |
| `--from`       | Start date (YYYY-MM-DD or `today`) |
| `--to`         | End date (YYYY-MM-DD or `today`) |
| `--limit`      | Maximum number of results |
| `--offset`     | Result offset for pagination |
| `-o, --output` | Output format |

<Tabs syncKey="time-list">
  <TabItem label="This week">
    ```bash
    redmine time list --user me --from 2024-01-15 --to 2024-01-19
    ```
  </TabItem>
  <TabItem label="By issue">
    ```bash
    redmine time list --issue 123
    ```
  </TabItem>
</Tabs>

## get

```bash
redmine time get <id> [flags]
```

Aliases: `show`, `view`.

## update

```bash
redmine time update <id> [flags]
```

| Flag | Description |
|------|-------------|
| `--hours`    | New hours |
| `--activity` | New activity (name or ID) |
| `--comment`  | New comment |
| `--date`     | New date (YYYY-MM-DD or `today`) |

## delete

<Badge text="destructive" variant="caution" />
```bash
redmine time delete <id> [flags]
```

| Flag | Description |
|------|-------------|
| `-f, --force` | Skip confirmation prompt |

## summary

```bash
redmine time summary [flags]
```

Displays a summary of time entries grouped by project and activity. Accepts the same filter flags as [`list`](#list).

<Aside type="tip">
  Combine `summary` with `--from today --user me` for a quick end-of-day breakdown of your own work.
</Aside>
